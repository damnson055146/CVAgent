工作原理

从目标到交付，仅需三步。

1

定义目标

用自然语言描述任务，选择合适的 Agent 模板。

2

编排流程

拖拽连接节点，设置工具与权限，预览输入输出。

3

一键运行

并行执行各 Agent，自动同步上下文并产出结果。

常见问题

是否支持离线/本地模型？
支持通过本地推理服务或 API 网关接入，包括多模型路由与回退策略。
如何限制危险操作？
通过权限清单、沙箱执行和审计日志实现可控边界，关键操作需显式批准。
能否与现有 CI/CD 集成？
提供 CLI 与 Webhook，支持 GitHub Actions、GitLab CI、Jenkins 等。
